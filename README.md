# 初一英语听力训练网页

专为**新牛津沪教版 七年级第一学期**设计的英语听力训练工具。

## 📋 功能特点

### 三大训练模块

1. **单词听写** - 60个核心词汇
   - 覆盖 Unit 1-6 重点单词
   - 支持速度调节（慢速/正常/快速）
   - 即时拼写检查和反馈
   - 自动记录错题

2. **句子理解** - 18道听力选择题
   - 真实情境句子练习
   - 选择题形式，即时反馈
   - 可重复播放句子
   - 答案解析

3. **短文听力** - 6篇短文理解
   - 每篇3个理解题
   - 涵盖日常、环保、季节等主题
   - 综合听力能力训练

4. **错题本**
   - 自动收录所有错题
   - 可重新播放错题音频
   - 支持清空记录

## 🚀 使用方法

### 方式一：直接打开（推荐）

1. 下载整个文件夹
2. 双击打开 `index.html` 文件
3. 开始练习！

### 方式二：本地服务器

如果浏览器有安全限制，可以使用本地服务器：

```bash
# 使用 Python 3
python -m http.server 8000

# 使用 Node.js (需安装 http-server)
npx http-server
```

然后在浏览器访问 `http://localhost:8000`

## 💡 使用小贴士

1. **首次使用**：第一次点击"播放"按钮可能需要稍等片刻，浏览器正在加载语音引擎

2. **听不清？**
   - 调慢播放速度至 0.75x
   - 点击"播放"按钮可以重复播放

3. **错题太多？**
   - 在错题本中重新练习
   - 多次播放加深记忆

4. **保存进度**
   - 进度自动保存，随时可以关闭
   - 点击"重新开始"会重置该模块进度

5. **最佳练习建议**
   - 每天坚持练习 15-20 分钟
   - 先从正常速度开始，熟练后尝试快速
   - 定期查看错题本，针对性复习

## ⚠️ 常见问题

**Q: 听不到声音怎么办？**  
A: 
- 确认浏览器允许播放音频
- 检查设备音量是否打开
- 尝试更换浏览器（推荐 Chrome）

**Q: 发音听起来很机械？**  
A: TTS 技术虽不如真人录音自然，但足够用于单词和句子练习。您可以配合教材音频一起使用。

**Q: 可以添加更多题目吗？**  
A: 可以！打开 `js/data.js` 文件，按照现有格式添加新的单词、句子或短文。

**Q: 进度突然丢失？**  
A: 清除浏览器缓存会导致进度丢失。建议定期截图记录进度。
